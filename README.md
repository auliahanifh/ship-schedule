Aplikasi ini dibuat menggunakan [Next.js](https://nextjs.org) dan postgreSQL untuk database, ditujukan untuk Pelindo Tanjung Perak Sub Regional III

## Tahapan pertama
Untuk uji coba pada development server, jalankan:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Kemudian, buka URL [http://localhost:3000](http://localhost:3000) pada web browser device, untuk melihat aplikasi.
